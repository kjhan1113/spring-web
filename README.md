### Intelli J ShortCut
- alt + insert : insert functions

- ctrl + W : extend selection
- ctrl + alt + L : reformat file
- ctrl + shift + T : create testcase class
- ctrl + shift + enter: move to next line
- ctrl + alt + V: extract variable
- ctrl + alt + M: extract method
